MK PUBG Mobile Tool v1.0.8
================================

INSTALLATION:
1. Extract all files to a folder
2. Run MK-PUBG-Mobile-Tool.exe

REQUIREMENTS:
- Windows 7/8/10/11
- Gameloop Emulator installed
- PUBG Mobile installed in Gameloop

FEATURES:
- Fix graphics settings for all PUBG Mobile versions
- Unlock FPS limits
- Change graphics styles
- Gameloop optimizer
- iPad view support

USAGE:
1. Open Gameloop and start PUBG Mobile
2. Run this tool
3. Click "Connect to Gameloop"
4. Select your desired graphics settings
5. Click "Submit"
