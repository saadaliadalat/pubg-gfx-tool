# EX Tool

EX Tool is a desktop utility for optimizing PUBG Mobile on Gameloop.

## Features
- Graphics quality and FPS controls
- Gameloop and PC optimization helpers
- Desktop shortcut creation
- DNS and iPad-view utilities

## Run from source
```bash
pip install -r requirements.txt
python main.py
```

## Build EXE
```bat
build_simple.bat
```

Output executable:
- `dist\\EX-Tool.exe`

## Note
This project is not affiliated with PUBG Mobile or Gameloop.
